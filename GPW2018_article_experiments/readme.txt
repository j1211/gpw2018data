GPW2018実験データ

GPW2018論文の対戦実験のデータです。

評価値は、整数値になるように適当倍してあります。(具体的にはweight = 0.001のケースにて, 1000倍しています）
青駒の個数 + 位置の評価 * 0.001
ではなくて、
1000 * 青駒の個数 + 位置の評価
とする感じ。

スクリーンショットと履歴ファイル(.txt)が主になります。

ファイルの番号と実験が対応しています。
以下はその説明です。

------------------------
実験1. どのAIが強い？

1〜12 : AI-A, AI-B, AI-C, AI-Dの総当りです。
・各組み合わせ先手後手入れ替えで50試合ずつ。（したがって全部で600試合あります）
・探索の深さは5
・初期配置は毎試合ランダムに決定
・300手で決着がつかない場合は引き分け
・評価値のパラメタ設定は：
	n = 1(各駒からゴールまでのマンハッタン距離の1乗和）
	weight = 0.001 (青駒の評価を優先）

AI-A … 最小値を手の評価 (MinMaxAI2)
AI-B … 平均値を手の評価 (AveScore2)
AI-C … 最善手多数決     (Tasuketu2)
AI-D … 紫駒             (Murasaki)

対戦組み合わせは, result○_先手_後手.pngを参照ください。
